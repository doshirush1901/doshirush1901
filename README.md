### What I'm Working On

**[Pantheon](https://github.com/doshirush1901/pantheon)** — An open-source AI sales agent framework.

One orchestrator. 17 specialist agents. Each named after a Greek god, each owning a different part of the revenue pipeline:

| Agent | Role |
|---|---|
| **Athena** | Orchestrator — routes every request to the right specialist |
| **Hermes** | Sales outreach — contextual drip campaigns, hyper-personalized emails |
| **Plutus** | Finance — order book, cashflow forecasts, revenue tracking |
| **Atlas** | Project management — every order, milestone, payment, document |
| **Sphinx** | Gatekeeper — asks clarifying questions before wasting tokens |
| **Nemesis** | Learning — ingests every correction, rewires the brain during sleep |
| **Iris** | Intelligence — real-time company news, industry trends, geopolitics |
| **Cadmus** | Marketing — case studies, LinkedIn posts, newsletters |
| + 9 more | Research, quality, vendors, forecasting, content scheduling... |

Built with GPT-4o, Qdrant, Neo4j, Mem0. Runs on Telegram, Email, and API. Has a nightly dream cycle, an immune system, and a growth hormone.

It started as an internal tool for Machinecraft. Now it's open source — configure it for your company in 10 minutes.

### About Machinecraft

We build industrial vacuum forming machines — the PF1, PF2, AM, ATF, IMG, and FCS series. Our machines shape plastic sheets into automotive parts, packaging, medical devices, EV battery enclosures, and more. Customers in 20+ countries.

I built Pantheon because I got tired of answering the same technical questions at 2 AM. Now the AI does it better than I did.
